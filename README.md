# GRAVITON Institute - Site Scaffold

Local development:

```bash
cd "d:/Graviton website"
npm install
npm start
```

The site uses the provided frames in `ezgif-15de25ff9b8f31e1-jpg/` as a canvas-played background animation. Configure SMTP via `SMTP_HOST`, `SMTP_USER`, `SMTP_PASS` env vars to enable real email sending.
