# Always Read Before Editing

Before making ANY change to this project, always read the following files first:

- `index.html` — full page structure, opening screen markup, sections
- `css/styles.css` — all styles including opening screen, layout, and animations
- `js/scripts.js` — opening animation logic and all interactive behavior

Do not assume the current state of these files from memory or prior context. Always read them fresh, then ask the user to confirm the change before applying it.
